# usage
1. run ``` pnpm install ```
2. 把每一个`src_`开头的文件夹改为`src`
3. 控制台运行 ```pnpm dev``` 即可


# 搭建官方本地文档
1. 下载官方relese 包
2. 使用本地静态服务器(`js` 使用`http-server` | `servez`,  `python`使用```python -m http.server```)
